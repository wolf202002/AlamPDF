INSTALLATION
────────────
1. Double-click: AlamPDF_Setup.exe
2. Follow the on-screen steps
3. Done — AlamPDF icon appears on your Desktop

WHAT GETS INSTALLED
───────────────────
• AlamPDF application
• All required packages (auto-installed)
• Desktop shortcut
• Start Menu entry
• Add/Remove Programs entry

REQUIREMENTS
────────────
• Windows 10 or 11 (64-bit)
• Python 3.8+ (auto-installed if missing)
• Internet connection during first install only

UNINSTALL
─────────
Settings → Apps → AlamPDF → Uninstall
(or Start Menu → AlamPDF → Uninstall)
